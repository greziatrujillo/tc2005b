# My Git Learning of branches

This repository is to demonstrate the use of branches and simplified gitflow.

## Labs Periodo 1
This branch takes in all the labs from the second period of the semester. This way, everything is organized and all the files will be within one area.
This also makes it easier to connect files, such as when labs call for files from a different lab. 

With the exception of lab-express, lab-js and the first two labs (HTML and CSS) which are in their own branch and are unable to be merged with this branch. HTML and CSS labs are in the development branch since they were merged from their original branches into that one out of curiosity and testing purposes.
